INSTRUÇÕES PARA O PROFESSOR:

COMANDOS

pra instalação: npm i

Em ordem: 

Migração: npx prisma migrate dev --name init
Gerar prisma: npx prisma generate 
Seed: npx prisma db seed



Para rodar o backend OU frontend é: npm run dev

Para resetar a db: npm run resetdb

Para gerar os graficos e testar os endpoints: npm run loadtest (TEM QUE ESTAR RODANDO O BACKEND)
